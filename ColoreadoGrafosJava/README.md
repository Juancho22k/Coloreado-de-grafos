# Coloreado de Grafos – BFS / DFS
### Proyecto Nro. 3 | Técnicas de Programación 3 | 2025-2

Esta es una aplicación en Java con interfaz gráfica JavaFX que visualiza e implementa el
coloreado de grafos usando los algoritmos BFS (Breadth-First Search) y
DFS (Depth-First Search).

Requisitos del Sistema:

- Java JDK 8 (recomendado para compatibilidad nativa con JavaFX).
- IDE recomendado: NetBeans 8.2.
- Sistema operativo: Windows, Linux o macOS.

Instalación:

1. Clonar el repositorio: https://github.com/Juancho22k/GraphColoringApp.git
2. Abrir el proyecto en NetBeans 8.2.
3. Limpiar y Construir (Clean and Build) para verificar dependencias de JavaFX.
4. Ejecutar el archivo GraphColoringApp.java.

Funciones del Programa:

- Gestión Dinámica: Creación de nodos y aristas mediante interacción directa con el mouse en el canvas.
- Algoritmos de Coloración: Ejecución de lógica basada en BFS y DFS.
- Modo Paso a Paso: Visualización animada del recorrido del algoritmo para fines educativos.
- Interfaz Interactiva: Paneles laterales para control de parámetros y visualización de la pila de pasos.

Validaciones:

- Evita la asignación de colores idénticos en nodos conectados.
- Control de nodos huérfanos durante la ejecución del algoritmo.
- Gestión de eventos para evitar errores al eliminar elementos en uso.
